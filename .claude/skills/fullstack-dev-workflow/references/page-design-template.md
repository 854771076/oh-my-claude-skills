|----------|----------|--------|----------|
| 初始状态 | 页面加载完成 | 【UI描述】 | 【数据描述】 |
| 加载状态 | 数据请求中 | 显示loading | 数据为空 |
| 成功状态 | 数据请求成功 | 【UI描述】 | 数据填充完成 |
| 空状态 | 数据为空 | 显示空状态提示 | 数据为空 |
| 错误状态 | 数据请求失败 | 显示错误提示 | 错误信息 |
| 权限不足状态 | 无访问权限 | 显示权限提示 | 无数据 |

### 4.2 状态流转图

```
初始状态 → 加载状态 → 成功状态/空状态/错误状态
                 ↓
           权限不足状态（若权限校验失败）
```
