## 二、核心开采系统

### 2.1 质押开采流程

用户质押LUCK代币进入"矿池"，每6小时可"开采"一次获得NFT盲盒：

```
开采流程：
1. 用户质押LUCK代币进入矿池
2. 选择锁定期（无锁定/7天/30天/90天）
3. 每6小时可开采一次
4. 开采获得NFT盲盒
5. 打开盲盒获得宝石NFT（按稀有度分布）
6. 稀有度越高，权益越大（分红权、概率加成等）
7. 多个低稀有NFT可合成升级为高稀有
```

### 2.2 稀有度分布

| 等级 | 名称 | 概率 | NFT权益 |
|------|------|------|---------|
| **普通** | 矿石碎片 | 60% | 可合成升级，无特殊权益 |
| **稀有** | 宝石矿晶 | 30% | +2%开采概率加成 |
| **传奇** | 王者宝石 | 8% | 手续费池0.5%月分红权 |
| **神话** | 星辰之心 | 2% | 手续费池2%月分红权 + 免手续费开采 |

### 2.3 质押分层设计

| 锁定期 | 稀有概率加成 | 开采奖励加成 | 退出手续费 |
|--------|------------|------------|----------|
| 无锁定 | 0% | 100%基础 | 3% |
| 7天 | +2% | 105% | 期满自动解锁 |
| 30天 | +5% | 115% | 期满自动解锁 |
| 90天 | +10% | 130% | 期满自动解锁 |

### 2.4 合成升级机制

| 合成路径 | 消耗 | 手续费 |
|---------|------|--------|
| 3个普通NFT → 1个稀有NFT | + 10 LUCK | 5% |
| 3个稀有NFT → 1个传奇NFT | + 50 LUCK | 5% |
| 3个传奇NFT → 1个神话NFT | + 200 LUCK | 5% |

**设计目的**：合成消耗代币，形成代币销毁路径，防止通胀。

## 三、NFT系统设计

### 3.1 宝石NFT（核心产出）

#### 3.1.1 NFT主题与风格

- **主题**：矿石/宝石矿工主题（与开采概念契合）
- **视觉风格**：像素风格（AI生成，成本低，Web3风格）
- **卡面元素**：稀有度标识、编号、宝石图像、名称、属性（硬度/光泽/纯度）、发现者、发现时间、权益图标

#### 3.1.2 NFT内容分类（10大类）

| 类别 | 示例名称 | 视觉风格 | 数量 | 获取方式 |
|------|---------|---------|------|---------|
| **基础矿石** | 铜矿碎片、铁矿碎片、石灰石 | 简单像素，灰褐色系 | 50种 | 开采(普通) |
| **稀有宝石** | 红宝石、蓝宝石、翡翠、琥珀 | 精致像素，多彩光效 | 20种 | 开采(稀有) |
| **传奇宝石** | 黑曜石之王、血钻之心、月光石 | 华丽像素，动态光效 | 10种 | 开采(传奇)/合成 |
| **神话宝石** | 星辰之心、创世之源、永恒钻石 | 终极像素，全屏特效 | 3种 | 开采(神话)/合成 |
| **特殊矿脉** | 月光矿脉、日耀矿脉、彩虹矿脉 | 特殊活动限定 | 10种 | 限时活动 |
| **合成产物** | 合成钻石、合成红宝石 | 合成专属标识 | - | 合成获得 |
| **纪念版** | 创世矿工纪念、周年庆典宝石 | 特殊事件纪念 | 不限 | 里程碑奖励 |
| **品牌合作** | 品牌定制宝石 | 品牌元素融入 | 定制 | 品牌活动 |
| **公会专属** | 公会徽章宝石、公会守护石 | 公会标识融入 | 定制 | 公会系统 |
| **成就奖励** | 首位矿工纪念、开采之王宝石 | 成就专属标识 | 定制 | 成就解锁 |

#### 3.1.3 AI生成流程

```
NFT生成流程：
1. 确定稀有度等级（根据概率分布）
2. 选择NFT类别（根据稀有度选择对应类别库）
3. AI生成像素图像（使用预设Prompt生成）
   Prompt模板："Pixel art [宝石名称], [稀有度描述], [颜色描述], [特效描述], 16x16/32x32 pixels, retro game style"
4. 生成唯一属性（硬度、光泽、纯度等随机数值）
5. 组装NFT元数据（整合图像、属性、权益信息）
6. 上链铸造（链上存储关键数据，图片存储在IPFS/Arweave）
```

## 四、PVP炸弹人系统

### 4.1 核心玩法机制

#### 4.1.1 放置炸弹

| 机制 | 详情 |
|------|------|
| **初始炸弹数** | 每个玩家初始可放置1个炸弹 |
| **爆炸时间** | 炸弹放置后2秒爆炸 |
| **爆炸范围** | 十字形（上下左右各2格） |
| **爆炸效果** | 可炸毁方块、玩家、怪物 |
| **炸弹升级** | 收集"炸弹强化宝石"可增加炸弹数量（上限5个）；收集"爆炸范围宝石"可增加爆炸范围（上限十字各5格） |
| **炸弹伤害** | 被炸中直接死亡（无护盾）；有护盾卡时免疫一次；被炸死后身上宝石全部掉落 |

#### 4.1.2 场内挖矿

| 机制 | 详情 |
|------|------|
| **挖矿方式** | 地图内分布矿石方块，使用镐子开采 |
| **挖矿结果** | 开采后获得宝石NFT（场内临时），自动存入玩家背包 |
| **矿石类型** | 普通矿石60%、稀有矿石30%、传奇矿石8%、神话矿石2%、特殊矿石（炸弹强化、爆炸范围等道具）随机 |
| **挖矿限制** | 每次开采消耗"开采次数"（门票决定总次数）；开采需要时间（2秒）；开采期间暴露，容易被击杀 |
| **镐子加成** | 使用镐子可提升稀有概率 |

#### 4.1.3 击杀夺宝

| 机制 | 详情 |
|------|------|
| **击杀方式** | 炸弹炸死其他玩家 |
| **击杀奖励** | 击杀后获得死者身上全部宝石；击杀者宝石数量暴增 |
| **击杀风险** | 击杀者身上宝石越多，越容易被反杀；击杀需要精准操作 |
| **策略博弈** | 击杀 vs 挖矿的选择；击杀目标需要策略判断（宝石多的玩家易击杀，宝石少的玩家难击杀） |

#### 4.1.4 负重减速机制

| 机制 | 详情 |
|------|------|
| **负重计算** | 宝石数量影响移动速度 |
| **基础速度** | 100% |
| **减速规则** | 每携带10宝石：移动速度-5% |
| **速度下限** | 30%（无法更慢） |
| **负重策略** | 携带大量宝石 = 高收益但易被击杀；携带少量宝石 = 快速移动但收益低 |
| **负重解除** | 死亡后宝石全部掉落，速度恢复；带出地图后宝石清空负重 |

### 4.2 门票系统

| 门票类型 | 价格 | 可入场时长 | 开采次数上限 | 炸弹数量上限 | 可带出宝石上限 |
|---------|------|-----------|------------|------------|--------------|
| **基础门票** | 50 LUCK | 5分钟 | 10次 | 1个 | 20个 |
| **高级门票** | 100 LUCK | 10分钟 | 20次 | 3个 | 50个 |
| **VIP门票** | 200 LUCK | 15分钟 | 30次 | 5个 | 100个 |

**门票池机制**：
- 所有门票LUCK进入门票池
- 可带出总宝石价值 = 门票池总额 × 70%
- 剩余30%进入累积池 + 项目方分成
- 每位存活玩家按带出宝石比例分配

### 4.3 地图与环境设计

| 元素 | 详情 |
|------|------|
| **地图大小** | 20×20格 |
| **可炸毁方块** | 50%（内部藏有宝石/道具） |
| **不可炸毁方块** | 20%（墙体，障碍物） |
| **空地** | 30%（玩家移动区域） |
| **怪物分布** | 普通矿工怪物5只、稀有宝石怪物3只、传奇宝石怪物1只 |
| **陷阱分布** | 地刺陷阱10个、火焰陷阱5个、冰冻陷阱5个、传送陷阱3个 |
| **动态缩圈** | 每2分钟地图边缘缩圈，缩圈区域内玩家强制死亡 |

**怪物AI等级**：
| 等级 | 行为 | 适用场景 |
|------|------|---------|
| Lv1 | 简单巡逻 | 新手局 |
| Lv2 | 主动追踪玩家 | 普通局 |
| Lv3 | 智能躲避炸弹 | 高手局 |

**陷阱效果**：
| 陷阱 | 效果 |
|------|------|
| 地刺陷阱 | 踩中减速50%持续5秒 |
| 火焰陷阱 | 踩中每秒伤害10%（持续燃烧） |
| 冰冻陷阱 | 踩中冻结3秒 |
| 传送陷阱 | 踩中传送至随机位置 |

### 4.4 玩家少时运作机制

| 机制 | 详情 |
|------|------|
| **怪物填充系统** | 玩家数量 < 4人时，自动生成怪物补充；怪物数量 = 4 - 玩家数量；怪物可击杀获得宝石，补偿玩家对抗缺失 |
| **单人挑战模式** | 玩家数量 = 1人时，自动切换为单人挑战模式；目标：在限定时间内收集最多宝石并存活；单人挑战奖励：存活奖励门票费用的50%返还、宝石奖励按价值兑换LUCK、成就奖励专属NFT；单人挑战难度：怪物数量增加（10只）、陷阱数量增加（20个）、缩圈速度加快 |
| **累积池吸引力机制** | 未被带出宝石90%进入累积池；累积池跨局累积，越来越大；累积池总额实时公示；累积池达到阈值时触发"累积池争夺赛"（门票免费，所有人可参与） |

### 4.5 胜利条件与奖励分配

| 条件 | 详情 |
|------|------|
| **胜利条件** | 存活即胜利（不比较宝石数量） |
| **存活玩家奖励** | 带出宝石按价值兑换LUCK；带出宝石NFT自动铸造为真实NFT |
| **奖励计算** | 每位存活玩家按带出宝石比例分配门票池 |
| **宝石稀有度系数** | 普通宝石1 LUCK、稀有宝石5 LUCK、传奇宝石20 LUCK、神话宝石100 LUCK |
| **未被带出宝石处理** | 90%进入累积池、10%归项目方（可动态调整） |
| **死亡玩家** | 宝石全部掉落、门票费用不返还、可选择重新购买门票入场 |

### 4.6 PVP策略博弈

| 策略类型 | 策略详情 | 优劣势 | 适用玩家 |
|---------|---------|---------|---------|
| **保守挖矿型** | 避开其他玩家，专注挖矿收集宝石，控制宝石数量保持移动速度 | 优势：安全稳定；劣势：收益较低，容易被击杀型玩家盯上 | 新手、保守玩家 |
| **激进击杀型** | 主动寻找其他玩家，使用炸弹击杀抢夺宝石，击杀后快速转移 | 优势：高风险高收益；劣势：需要操作技巧，容易被反杀 | 高手、激进玩家 |
| **怪物猎杀型** | 专注击杀怪物获得宝石，避开玩家对抗 | 优势：收益稳定，风险可控；劣势：收益上限较低 | 玩家少时、中级玩家 |
| **陷阱利用型** | 利用陷阱困住其他玩家，引诱玩家踩陷阱后击杀 | 优势：创新玩法，出奇制胜；劣势：需要熟悉地图陷阱分布 | 地图熟悉玩家、策略型玩家 |

## 五、代币经济学设计（LUCK代币）

### 5.1 代币价值锚定（五重价值支撑）

| 价值来源 | 具体体现 |
|---------|---------|
| **功能价值** | 开采必需品、合成消耗、高级门票（四重功能需求） |
| **分红价值** | 持币分红 + 质押分红（持续现金流） |
| **治理价值** | DAO投票权、参数调整权、发展方向权 |
| **稀缺价值** | 总量1亿固定上限 + 每日回购销毁 |
| **情绪价值** | 社区认同、排行榜特权、身份象征 |

### 5.2 代币发行结构（总量1亿枚固定上限）

| 分配类别 | 数量 | 占比 | 释放规则 |
|---------|------|------|---------|

| 社区空投 | 1500万 | 15% | 任务参与领取，分批次释放 |
| 质押奖励池 | 4000万 | 40% | **永不增发**，每日开采释放 |
| 团队/顾问 | 1000万 | 10% | 锁仓12个月 + 24个月线性释放 |
| 生态基金 | 1000万 | 10% | 合作激励、市场推广 |
| 流动性池 | 500万 | 5% | DEX初始流动性，锁定6个月 |

### 5.3 代币流通控制（四重锁定机制）

| 锁定来源 | 锁定比例 | 锁定效果 |
|---------|---------|---------|
| 团队代币 | 10% | 锁仓12个月，后续线性释放 |
| 质押池 | 动态（用户行为） | 质押期间不可流通 |
| 合成消耗 | 每次合成消耗LUCK | 销毁路径，减少供应 |
| 回购销毁 | 每日20%手续费回购 | 永久销毁，不可恢复 |

| 时间 | 初始流通 | 质押锁定 | 回购销毁 | 实际流通 |
|------|---------|---------|---------|---------|

**关键洞察**：实际流通量持续下降 → 强通缩效应

### 5.4 手续费收入分配（六方平衡）

### 5.5 代币分红机制（差异化收益）

**分红池结构**：
- 持币分红池（总手续费7.5%）：所有持币者按持币量分配，无需质押，被动收益
- 质押分红池（总手续费17.5%）：仅质押者按质押量×时间系数分配，收益≈持币分红4倍

**时间系数设计**：
| 锁定期 | 时间系数 | 分红加成 |
|--------|---------|---------|
| 无锁定 | 1.0 | 基础分红 |
| 7天 | 1.1 | +10%分红 |
| 30天 | 1.3 | +30%分红 |
| 90天 | 1.6 | +60%分红 |

**收益对比测算**（假设手续费月收入10000 USDC）：
| 用户持币 | 状态 | 分红来源 | 月收益 |
|---------|------|---------|-------|
| 10000 LUCK | 仅持币，不质押 | 持币分红池 | ~7.5 USDC |
| 10000 LUCK | 质押30天 | 质押分红池 | ~32 USDC |
| 10000 LUCK | 质押90天 | 质押分红池 | ~40 USDC |

### 5.6 代币治理权（分阶段开放）

**治理开放时间线**：
| 阶段 | 时间范围 | 治理规则 |
|------|---------|---------|
| 项目方主导 | 0-6个月 | 项目方拥有100%治理权 |
| 有限治理 | 6-12个月 | 持币 >5000 LUCK 可投票；权重 = 持币量 × 持币时间系数；项目方保留50%权重 |
| 完全治理 | 12个月后 | 所有持币者可投票；权重 = 持币量 × 持币时间系数；项目方权重降至10% |

**持币时间系数**：
| 持币天数 | 时间系数 | 投票权重加成 |
|---------|---------|------------|
| <30天 | 1.0 | 基础权重 |
| 30-90天 | 1.2 | +20%权重 |
| 90-180天 | 1.5 | +50%权重 |
| >180天 | 2.0 | +100%权重 |

**治理范围**：
| 阶段 | 可治理事项 |
|------|-----------|
| 阶段1 | 无 |
| 阶段2 | 开采奖励调整、手续费调整（±10%范围） |

### 5.7 代币销毁机制（通缩核心）

| 销毁来源 | 月销毁量（估算） | 年销毁量（估算） |
|---------|-----------------|-----------------|
| 合成消耗 | ~50000 LUCK | ~60万LUCK |

### 5.8 防雪崩机制总结

| 机制 | 作用 | 效果 |
|------|------|-----|
| 总量固定上限 | 供应上限 | 永不增发，稀缺保障 |
| 质押锁定 | 减少流通 | 大量代币被锁定，减少抛压 |
| 分阶段解锁 | 防止挤兑 | 团队代币锁仓12个月 |
| 回购销毁 | 永久买盘 | 每日回购形成价值支撑 |
| 持币分红 | 激励持有 | 持币有收益，减少抛售意愿 |
| 质押分红加成 | 强烈激励质押 | 质押收益4倍持币收益 |
| 开采量熔断 | 防止恶意开采 | 单日开采人次>500时次日奖励降低10% |
| 解锁熔断 | 防止挤兑式抛售 | 单日解锁质押量>总池5%时分3天释放 |
| 治理时间权重 | 激励长期持有 | 持币时间越长投票权重越大 |

| **技术风险** | 合约漏洞 | 多签托管 + 定期审计 |
